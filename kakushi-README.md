# 隠し字クイズ — KAKUSHI

漢字・熟語・四字熟語を8×8グリッドから当てるクイズゲームです。

## ファイル構成

```
kakushi/
├── index.html   # メインHTML・UI
├── words.js     # 単語データベース（漢字・熟語・四字熟語）
├── game.js      # ゲームロジック
└── README.md    # このファイル
```

## ローカルで動かす

ファイルをダウンロードして `index.html` をブラウザで開くだけで動作します。

## Renderでのデプロイ方法

→ 別途 `DEPLOY.md` を参照してください。
